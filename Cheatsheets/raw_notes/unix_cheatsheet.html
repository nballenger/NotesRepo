Unix Cheat Sheet:

Unix Commands:
command [flags/options] [target_of_command] Basic command syntax
command1;command2 Enter multiple commands per line
Getting Help:
command (--help | -h) Typically gets the command to give you some help text
man command Opens a manual page if there is one
Moving Around the Command Line:
ctrl-a Take cursor to beginning of line
ctrl-e Take cursor to end of line
ctrl-k Delete from cursor to end of line
ctrl-c Sends SIGINT to current process
ctrl-u Delete entire current line
ctrl-s Pauses current writing to STDOUT
ctrl-q Resumes writing to STDOUT from ctrl-s freeze
ctrl-d Signals end of input for some programs (cat, e.g.)
ctrl-z background the current process
Moving Around the Filesystem:
cd PATH Change Directory
pwd Present Working Directory
ls [-lahS] List directory contents
cd ~ Go to home directory
Getting General Information:
date Print date and time
who List logged in users
whoami Current username
who am i Extended login information
env Prints current environment variables
echo $VAR Print current value of $VAR environment variable
Managing Jobs:
ctrl-z background the current process
jobs list current background processes
fg number foregrounds the numbered process
Filesystem Wildcards:
* Any number of any character
? Any single character
[] Any of enclosed characters, once
{n,m} At least n but no more than m of preceding char
Finding Things:
which commandname Finds command if it's in $PATH
locate string Finds string in paths across filesystem
find [path...] [expression] More powerful search tool--see 'man find'
Unix Users:
  • Every user has a username, most have a home directory
  • Home directories are in /home on *nix systems, /Users on OS X
  • Users belong to at least one group, usually the group with their name
  • Groups are used to set permissions across multiple users
  • You can see the users and groups in /etc/passwd and /etc/group
  • Users have a uid number, groups have a gid number
File Permissions:
Example Permissions String:
drwxr-xrw-
Byte 1: Directory/File/Link
Bytes 2-4: Owner Permissions
Bytes 5-7: Group Permissions
Bytes 8-10: World Permissions
Changing File Permissions and Ownership:
chmod [options string] file Modify the permissions of a file
chown [username|uid] file Modify ownership of a file
chgrp [groupname|gid] file Modify group of a file
Changing your Password:
passwd Will prompt for old and new password
Working with Files:
cp original_file copied file Copy
mv original_file new_file Move
rm filename Remove
mkdir dirname Make Directory
rmdir Remove Directory
rm -Rf dirname Remove (with Recursive and Force flags--USE W CAUTION)
touch filename Create empty file / update timestamp of old file
tar -cvf archivename.tar file Create a tape archive of a file or directory
tar -xvf archivename.tar Un-archive a tape archive
tar -czvf archivename.tar.gz file Create a gzipped tape archive of a file or dir
tar -xzvf archivename.tar.gz Un-zip and un-archive a gzipped archive
zip filename.zip file Create a zip file archive
unzip filename.zip Unzip a zip archive
Viewing File Contents:
less filename less File Reader
more filename more File Reader
cat filename Dump file contents to STDOUT
Manipulating Streams:
wc file Word Count (returns char, word, line counts)
sort file Sorts contents of file
uniq file Returns unique lines in file
grep "pattern" file Regular Expression Search
sed 'sed commands' file Multi-function Stream Editor
tr SET1 SET2 Translate one set of characters to another
awk '{awk commands}' file Multi-function command language
Viewing System Information:
top Displays current processes and resource use
ps aux Displays current processes
jobs Displays currently running shell commands
df Displays Disk Usage of currently mounted vols
du Displays Disk Usage of current dir (recursively)
du -s Displays Disk Usage of current dir (summary)
Changing Environment Variables:
VAR="value" <enter> Temporarily
Add to ~/.bash_profile For all sessions
Adding Environment Variables to .bash_profile or .bash_rc:
export VAR="value" Sets variable for next time shell starts up
Adding command aliases to .bash_profile or .bash_rc:
alias aliasname="command"
Forcing your shell to reload with a new configuration:
source ~/.bash_profile
Input/Output Redirection
  • All unix commands should take streams as input, give streams as output
  • One command's output should be able to go into the input of another command
  • Streams are directed starting with STDIN
  • Streams are output to STDOUT unless otherwise specified
Available IO Manipulators:
command > outputfile Write to file instead of STDOUT (clobbers file)
command >> outputfile Append to file instead of print to STDOUT
command1 | command2 Take output of cmd1, put into input of cmd2
command < inputfile Take inputfile as stream, instead of STDIN
Example Pipeline:
cat filename | grep "MI" | awk "{print $3}" | sed 's/[^0-9]//g' | sort | uniq > MI_phones.txt
