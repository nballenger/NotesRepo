Chapter 1: Running a CoffeeScript Program
  * Install Node.js and npm
  * Install coffeescript with:

    npm install -g coffee-script
    coffee -v

  * Open the console with 'coffee', exit with ctrl-d
  * Write to files with .coffee extension
  * Compile files with:

    coffee -c myfile.coffee
