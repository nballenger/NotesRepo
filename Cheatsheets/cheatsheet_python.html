#!/usr/bin/env python

message = "Hello, World!"
print message

Save as helloworld.py, run with python ./helloworld.py.

Developed starting in the late 1980's by Guido van Rossum. The 2.0 release in 2000 brought in automated garbage collection and unicode support, and the 3.0 release in 2008 added features and refinements, many of which were also backported to 2.6 and 2.7. Python has a rather specific design philosophy, typified by statements like, Explicit is better than implicit. and Simple is better than complex. See PEP 20 for the original "Zen of Python" document.

One way to think about the particularities of python is that it's in many ways diametrically opposed to perl. Where perl was created by a linguist, python is the creation of a mathematician. Perl has "more than one way to do it," while python generally strives to have a single, canonical way of expressing specific operations. Some of the more striking features of python are:

  • White space is meaningful and proper formatting is enforced by the interpreter—consequently python programs tend to be very readable (though not friendly to careless copy-paste jobs).
  • Set operations are built into the language, so performing unions and intersections, as well as set comparisons, is syntactically simple.
  • Single statement, anonymous functions (lambdas) can act as function literals, allowing a (limited) form of functional programming.
  • List and dictionary 'comprehensions' allow quick filtering / mapping of composite variables.

Operators

Arithmetic

Operator Description Example
+ Return the sum of the left and right operands. 
10 + 5  # evaluates to 15
- Return the difference of the left operand minus the right operand. 
20 - 2  # evaluates to 18
* Return the product of the left and right operands. 
20 * 2  # Evaluates to 40
/ Return the quotient of the left operand divided by the right operand. 
20 / 3    # Integer division: evaluates to 6
20.0 / 3  # Float division: evaluates to 6.6666666...
% Return the remainder of the left operand divided by the right operand. 
20 % 3  # Evaluates to 2
** Return the product of the left operand raised to the power of the right operand. 
2 ** 5  # Evaluates to 32
// Return the integer floor of the quotient of the left operand divided by the right operand. 
20 // 3    # Evaluates to 6
20.0 // 3  # Evaluates to 6

Comparison

Operator Description Example
== Return the boolean value expressing the equality of the value of the left operand and the value of the right operand. 
1 == 1  # Evaluates to true
1 == 0  # Evaluates to false
!= Return the boolean value expressing the inequality of the value of the left operand and the value of the right operand. 
1 != 1  # Evaluates to false
1 != 0  # Evaluates to true
<> Obsolte synonym for !=.
> Return the boolean value expressing whether the left operand is greater than the right operand. 
1 > 0  # Evaluates to true
0 > 1  # Evaluates to false

Assignment

Operator Description Example
= Set left operand to value of right operand. 
a = 10
b = 15
c = a + b  # c set to 25 (10+15)
+= Set left operand to its current value plus the value of the right operand. 
a = 10
a += 10  # Addition: a set to 20 (10+10)

b = 'abc'
b += 'd' # String concatenation: b set to 'abcd'
-= Set left operand to its current value minus the value of the right operand. 
a = 10
a -= 5  # a set to 5 (10-5)
*= Set left operand to its current value times the value of the right operand. 
a = 10
a *= 3  # a set to 30 (10*3)
/= Set left operand to its current value divided by the value of the right operand. 
a = 10
a /= 5   # Integer division: a set to 2 (10/5)
b = 10.0
b /= 3   # Float division: b set to 3.33333...
%= Set left operand to its current value modulo the value of the right operand. 
a = 10
a %= 3  # a set to 1 (10%3)
**= Set left operand to its current value to the power of the right operand. 
a = 10
a **= 2  # $a set to 100 (102)
//= Set left operand to the integer floor of the quotient of the left operand divided by the right operand. 
a = 10
a //= 3   # a is set to 3
b = 10.0
b //= 3   # b is set to 3
