Chapter 1: Systems of Linear Equations and Matrices

1.1 Introduction to Systems of Linear Equations

  • Equation of a line in a rectangular xy-coordinate system: ax + by = c
  • Linear Equation - An equation in the n variables x1, x2, ..., xn that can be expressed in the form a1x1 + a2x2 + ... + anxn = b, where the a and b terms are constants, and the a terms are not all zero.
  • System of linear equations, or linear system - A finite set of linear equations.
  • Solution of a linear system - a sequence of n numbers s1, s2, ..., sn for which the substitution x1 = s1, x2 = s2, ..., xn = sn makes each equation in the system a true statement.
  • Ordered n-tuple - format for a solution to a system of linear equations of the form (s1, s2, ..., sn)
  • Ordered pair - ordered tuple where n = 2
  • Ordered triple - ordered tuple where n = 3
  • Linear systems of two unknowns (x,y) graph lines in the x,y plane, and each solution to the system corresponds to a point of intersection of the lines. There are three possibilities:
    1. The lines may be parallel and distinct, having no intersection and no solution.
    2. The lines may intersect at one and only one point, giving exactly one solution.
    3. The lines may coincide, giving infinitely many points of intersection and solutions.
  • Consistent linear system - A system with at least one solution.
  • Inconsistent linear system - A system with no solutions.
  • The same is true for planes in a system of three unknowns:
    1. No solutions:
      • Three parallel planes, thus no common intersection
      • Two parallel planes, and no common intersection
      • Two coincident planes parallel to the third, no common intersection
    2. One solution, where the intersection of planes is a single point
    3. Infinitely many solutions:
      • The intersection of three planes is a line, like the spine of a book
      • Planes are all coincident
      • Two coincident planes intersecting the third, making a line.
  • Every system of linear equations (of any number of unknowns) has zero, one, or infinitely many solutions.
  • As a system of linear equations grows, it becomes increasingly complex to represent as formulae. You can represent it as a matrix of coefficients: a 1⁣1 a 1⁣2 ⋯ a 1⁣n a 2⁣1 a 2⁣2 ⋯ a 2⁣n ⋮ ⋮ ⋱ ⋮ a m⁣1 a m⁣2 ⋯ a m⁣n which is called the augmented matrix for the system.
  • There are several elementary row operations that can be performed on a matrix:
    1. Multiply an equation / row through by a nonzero constant
    2. Interchange two equations / rows
    3. Add a constant times one equation / row to another
