Definitions
scalar a quantity completely determined by a single numerical value consisting of a signed 'magnitude'
vector a quantity determined by a direction and a maginitude; a 'directed' quantity represented by an arrow
magnitude the scalar, non-directional value of a vector, denoted |v|
zero vector,
null vector		 a vector with magnitude 0 and no direction, denoted 0
displacement vector given a point P, the vector which moves to a new location P'
unit vector a vector with a magnitude of 1
initial point for a line P1P2, P1 is the initial point
terminal point for a line P1P2, P2 is the terminal point
representative of if vector v moves P to P', and v has the same direction and magnitude as PP', then PP' is representative of v, though PP' is not the same as v, because v has no inherent initial point
localized vector,
directed segment		 a pair of points (P1P2) that implements / represents a vector
equidirectional,
parallel		 two non-zero vectors v1 and v2 that have the same direction
opposite,
antiparallel		 two non-zero vectors v1 and v2 that have opposite directions
collinear vectors that are parallel or antiparallel
vector addition P1P2 + P2P3 = P1P3, representing v1 + v2 = v3
vector subtraction v1 - v2 = v1 + (-v2)
vector multiplication by a scalar v ≠ 0 ∧ α > 0 ⇒ αv = vector(direction=v.direction, magnitude=α|v|)
v ≠ 0 ∧ α < 0 ⇒ αv = (-a)(-v) = vector(direction=-v.direction, magnitude=-α|v|)
resultant vector the vector which results from the addition of two vectors
triangle rule P1P2 + P2P3 = P1P3, representing v1 + v2 = v3
polygon rule the triangle rule generalized to a series of n vectors; if P1Pn = 0, polygon is closed
position vector,
bound vector		 the vector represented by the line OP in E3 from the origin O to a point P
free vector,
orientation vector		 any unbound vector that is free to be arbitrarily translated
vector equation of a line For P1 and P2 on a line, with vectors p1,p2, relative to some origin O, P1P2 = P1O + OP2, which is equal to the direction vector d = (-p1) + p2 = p2 - p1
parametric vector equation of a line For a point P with position vector p, and a line P1P2, P1P represents r = p-p1, so P is on the line iff p-p1 = t(p2-p1) for some scalar t; i.e.: p = (1-t)p1 + tp2, which is the parametric vector equation of the line relative to the origin O
linearly dependent if a is a non-zero vector, any vector v which is collinear to a is of the form αa for some scalar α.
    a ⇈ v ⇒ α = |v| / |a|
    a ⇅ v ⇒ α = -|v| / |a|
    v = 0 ⇒ α = 0
If v is of the form αa, v is linearly dependent on a. When a ≠ 0, vi is linearly dependent iff v ‖ a
Any vector v coplanar with a,b is linearly dependent on a,b
Any vector v is linearly dependent on a,b,c iff ∃α,β,γ such that v = αa + βb + γc
Any vector in E3 is linearly dependent on i,j,k
linearly independent In general, given a set of n non-zero vectors {vi}ni=1 and a set of scalars {αi}ni=1, there exists a linear combination of those vectors that vanishes / is equal to 0:
    α1v1 + α2v2 + ⋯ + αnvn = 0
where {αi}ni=1 are scalars.

If they only vanish when all scalars = 0, they are linearly independent.
coplanar given a vector v (OV) on plane AOB, v is coplanar with a and b iff ∃α,β such that v = αa + βb
linear combination for v that is coplanar with and b, v is a linear combination of a and b via v = αa + βb
vector basis Any ordered set of three non-coplanar vectors used for inferring a system of other vectors in E3
components of a vector for a vector v relative to a basis a,b,c, α,β,γ are the components of v that express it relative to a,b,c. In a Cartesian basis, denoted [vx,vy,vz]
scalar multiplication of a vector For v with components [α,β,γ], multiplication by a scalar s is given by
    s[α,β,γ] = [sα,sβ,sγ]
vector addition Given v1,v2, [α1,β1,γ1], [α2,β2,γ2] relative to basis a,b,c:
    v1 ± v2 = [α1,β1,γ1] ± [α2,β2,γ2] = [α1 ± α2, β1 ± β2, γ1 ± γ2]
vector equality v1 = v2 if [α1,β1,γ1] = [α2,β2,γ2]
transpose of a vector turning a vector into a 1x3 matrix: [⋯]T = [ ⋮ ]
orthogonal bases of a vector space that are mutually perpendicular
orthonormal mutually perpendicular bases that are also unit vectors
left handed vector space any system where, looking down from above, you see BLA
right handed vector space any system where, looking down from above, you see ALB
Cartesian basis an orthonormal, right-handed basis

Axioms and Rules of Vector Algebra

|v| ≥ 0 ⇔ v ≠ 0

(P1P2) = -(P2P1) = v = -(-v)

∀v ⇈ 0

  • Pertaining to multiplication of a vector by a scalar:
    • A: Existence of the vector by scalar product: αv is a vector
    • A: Existence of the zero element: 0v = v0 = 0
    • A: Existence of the neutral element: 1v = v1 = v
    • R: Associative law: α(βv) = (αβ)v
    • R: Distributive law: (α+β)v = αv + βv, α(v1 + v2) = αv1 + αv2
  • Pertaining to addition of vectors:
    • A: Existence of the vector sum: v1 + v2 is a vector
    • A: Existence of the neutral element: 0 + v = v + 0 = 0
    • A: Existence of the inverse element: v + (-v) = (-v) + v = 0
    • R: Commutative law: v1 + v2 = v2 + v1
    • R: Associative law: (v1 + v2) + v3 = v1 + (v2 + v3)
    • R: Change of direction rule: -(v1 - v2) = v2 - v1

For v = [vx,vy,vz] in i,j,k space, let Vxy be the foot of the perpendicular of point V onto plane XOY. From this:

    |v|2 = |OV|2 = |OVxy|2 + vz2 -- however:

    |OVxy|2 = vx2 + vy2
    ∴  |v|2 = vx2 + vy2 + vz2
    ∴  |v| = √vx2 + vy2 + vz2
