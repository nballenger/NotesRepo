Math

Size of sample space with N possible outcomes 2N
P{sample space} P{Ω} = 1
Probability of the empty set P{∅} = 0
Probability of event E made of outcomes {ω1, ω2, ω3, ...} P{E} = for ωk ∈ E, Σ P{ωk} = P{ω1} + P{ω2} + P{ω3} ...
Probability of a 2-way Union P{A ⋃ B} = P{A} + P{B} - P{A ⋂ B}
Probability of a 3-way Union P{A ⋃ B ⋃ C} = P{A} + P{B} + P{C} - P{A ⋂ B} - P{A ⋂ B} - P{B ⋂ C} + P{A ⋃ B ⋃ C}
Probability of a complement: P{AC} = 1 - P{A}
Probability of independent events: P{E1 ⋂ ... ⋂ En} = P{E1} * ... * P{En}
Permutations with replacement: Pr(n,k) = n * n * ... * n = nk
Permutations without replacement: P(n,k) = n(n-1) * (n-2) * ... (n-k+1) = n!/(n-k)!
Combinations with replacement: Cr(n,k) = (k+n-1k) = (k+n-1)! / k!(n-1)!
Combinations without replacement (n choose k): C(n,k) = (nk) = P(n,k)/P(k,k) = n! / k!(n-k)!
Conditional probability of A given B P{A | B} = P{A ⋂ B} / P{B}
Intersection, general case P{A ⋂ B} = P{B} * P{A | B}
Probability mass function P(x) = P{X = x}
Cumulative distribution function F(x) = P{X <= x} = Σ P(y) ∀ y <= x

Terms:

  • An experiment or trial produces a result or outcome.
  • A set (1 to N) of outcomes is an event.
  • The collection of all possible results is a sample space.
  • Events A and B are disjoint if A ⋂ B = ∅.
  • Events A1, A2, A3,... are mutually exclusive or pairwise disjoint if any two of the events are disjoint.
  • Events A, B, C, ... are exhaustive if their union equals the entire sample space.
  • Events E1, ..., En are independent if the occurrence of one event does not affect the probabilities of others.
  • Sampling with replacement means every sampled item is replaced into the initial set, so that any of the objects can be selected with probability 1/n at any time. In particular, the same object may be sampled more than once.
  • Sampling without replacement means that every sampled item is removed from further sampling, so the set of possibilities reduces by 1 after each selection.
  • Objects are distinguishable if sampling of exactly the same objects in a different order yields a different outcome, that is, a different element of the sample space. For indistinguishable objects, the order is not important, it only matters which objects are sampled and which ones aren't. Indistinguishable objects arranged in a different order do not generate a new outcome.
  • Conditional probability of event A given event B is the probability that A occurs when B is known to occur.
  • A random variable is a function of an outcome--a quantity that depends on chance.
  • A collection of all the probabilities related to X is the distribution of X. The function P(x) = P{X = x} is the probability mass function (pmf). The cumulative distribution function (cdf) is defined as F(x) = P{X <= x} = sum(P(y) for all y lte x).
  • The set of possible values of X is the support of the distribution F.
