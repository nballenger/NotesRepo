Chapter 2

Sets and Mappings

  • A 'mapping' is analogous to a function. Considered in terms of sets, you can say that a function may act on one set (inputs) to produce a second set (outputs). The inputs are the 'domain' of the function, the outputs are the 'range' or 'target'.
  • There may be inverse functions or mappings. For some function f(a) → b, the inverse function f-1 is that which fulfills g(b) → a.
  • A function and its inverse together are a 'bijection'. Bijections map every a in A to a unique b in B, and for every b in B there is exactly one a in A such that f(a) = b.
  • Functions that are not bijections have no inverse.
  • An interval is a set of numbers between endpoints. If the endpoints are included in the set it is a 'closed interval', if they are excluded it is an 'open interval.' A single interval may have a closed and an open endpoint, or both closed/open.
  • Intervals are useful in conjunction with the set operations intersection, union, and difference.
  • Logarithms are often useful in equations with exponential terms.
  • Every logarithm has a base a, which equates for a term x to 'the exponent to which a must be raised to get x.'

Solving quadratic equations

  • A quadratic equation has the form Ax2 + Bx + C = 0, where x is a real unknown, and A, B, C are constants.
  • Quadratic equations produce parabolas when plotted.

Trigonometry

Vectors

  • Only the lowest level vector operations should be aware of their numeric representation--otherwise always treat vectors as objects.
  • A 2D vector can be written as a combination of any two nonzero vectors which are not parallel. This property of the two vectors is 'linear independence.' Two linearly independent vectors form a 2D 'basis', which makes them 'basis vectors.'
  • Cartesian coordinates are vectors via a system of orthonormal basis vectors.
  • Any set of vectors form an orthonormal basis provided they are orthogonal and are each of unit length.
