Sets

Definitions

Term Definition
set A collection with clear and unambiguous rules for membership, in which order and multiplicity has no significance.
element / member An object in a set.
subset For sets A and B, A is a subset of B if every element in A also appears in B.
superset If A is a subset of B, then B is a superset of A.
proper subset A is a proper subset of B if it is both a subset of B and not equal to B.
proper superset B is a proper subset of A if it is both a superset of A and not equal to A.
null or empty set The empty set contains no elements, and is denoted ∅.
universal set The universal set is the superset of all elements under consideration, and is denoted U.
comparable sets when A is a proper superset of B (A ⊃ B) or B is a proper superset of A (B ⊃ A)
disjoint sets when A and B have no common elements
closure A set S is closed under operation x if and only if the result of x on two elements of S is also an element of S.

Operations

Equality

A = B A is equal to B if every element that appears in A also appears in B, and every element that appears in B also appears in A.

Union

A ⋃ B = { x : x ∈ A ∨ x ∈ B } The union of A and B is made of every element that occurs in either A or B. Therefore:
    ∴ A ⋃ B = B ⋃ A     …the union operation is commutative, and
    ∴ A ⊆ A ⋃ B     …A is a subset of the union of A and B, and
    ∴ B ⊆ A ⋃ B     …B is a subset of the union of A and B.

Generalized Union
⋃ni=1Si = S1 ⋃ S2 ⋃ S3 ⋃ ⋯ ⋃ Sn A generalized union of sets is made of every element that appears in any of the member sets.

Intersection
A ⋂ B = { x : x ∈ A ∧ x ∈ B } The intersection of A and B is made of every element that occurs in both A and B. Therefore:
    ∴ A ⋂ B = B ⋂ A     …the intersection operation is commutative, and
    ∴ (A ⋂ B) ⊆ A     …the intersection of A and B is a subset of A, and
    ∴ (A ⋂ B) ⊆ B     …the intersection of A and B is a subset of B.

Generalized Intersection
⋂ni=1Si = S1 ⋂ S2 ⋂ S3 ⋂ ⋯ ⋂ Sn A generalized intersection of sets is made of every element appearing in all of the member sets.

Difference
A - B = { x : x ∈ A ∧ x ∉ B } The difference of A and B is made of every element which appears in A and not in B. Therefore:
    ∴ (A - B) ⊆ A     …the difference of A and B is a subset of A, and
    ∴ (A - B) ⋂ (A ⋂ B) = ∅     …the difference of A and B, when intersected with the intersection of A and B, produces the null set, and
    ∴ (B - A) ⋂ (A ⋂ B) = ∅     …the difference of B and A, when intersected with the intersection of A and B, produces the null set, and
    ∴ (B - A) ⋂ (A - B) = ∅     …the difference of B and A, intersected with the difference of A and B, produces the null set.

Cartesian Product
A × B = { (a,b) | a ∈ A ∧ b ∈ B } The cartesian product of A and B is an ordered series of pairs (a,b), where element a is drawn from A and element b is drawn from B.

N-ary Cartesian Product
∏ni=1Si = S1 × S2 × ⋯ × Sn = { (s1,s2,⋯,sn) | s1 ∈ S1 ∧ s2 ∈ S2 ∧ ⋯ ∧ sn ∈ Sn } The n-ary cartesian product of a series of sets is an ordered series of tuples of length n, where each element is drawn from the input sets in order.

Symmetric Difference
A Δ B = { x : (x ∈ A ∧ x ∉ B) ∨ (x ∉ A ∧ x ∈ B) } The symmetric difference of A and B is made of all elements which appear in A but not B, or which appear in B but not A or, stated differently
A Δ B = (A ⋂ B') ⋃ (A' ⋂ B) …the union of A intersected with the complement of B and the complement of A intersected with B, or
A Δ B = (A - B) ⋃ (B - A) …the union of the difference of A and B with the difference of B and A, or
A Δ B = (A ⋃ B) - (B ⋂ A) …the difference between the union of A and B and the intersection of B and A. Therefore:
    ∴ A Δ B = B Δ A     …the symmetric difference operation is commutative, and
    ∴ (A Δ B) Δ C = A Δ (B Δ C)     …the symmetric difference operation is associative, and
    ∴ A ⋂ (B Δ C) = (A ⋂ B) Δ (A ⋂ C)     …the intersection operation is distributive over symmetric difference, and
    ∴ A Δ ∅ = A     …the symmetric difference of a set with the null set is the set itself, and
    ∴ A Δ A = ∅     …the symmetric difference of a set with itself is the null set.

Complement
A' = Ac = U - A = { x | x ∈ U ∧ x ∉ A } The complement of a set A is made of all elements in the universal set which do not appear in A itself. Therefore:
    ∴ A ⋃ A' = U     …the union of a set and its complement is the universal set, and
    ∴ A ⋂ A' = ∅     …the intersection of a set and its complement is the null set.

Theorems on Comparable Sets

For the following, if A is a subset of B, then:
A ⊆ B ⇒ A ⋂ B = A     …the intersection of A and B is equal to A, and
A ⊆ B ⇒ A ⋃ B = B     …the union of A and B is equal to B, and
A ⊆ B ⇒ B' ⊆ A'     …the complement of B is a subset of the complement of A, and
A ⊆ B ⇒ A ⋃ (B - A) = B     …the union of A with the difference of B and A is equal to B, and
A ⊆ B ⇒ (B - A) = (B ⋂ A')     …the difference of B and A is equal to the union of B and the complement of A.

Rules of Algebra for Sets

Given universe U and subsets A, B, and C, and rules ⋃, ⋂, and S':

Identity Rules
A ⋃ ∅ = A The union of a set A with the empty set is A.
A ⋂ U = A The intersection of a set A with the universal set is A.
A ⋃ U = U The union of a set A with the universal set is the universal set.
A ⋂ ∅ = ∅ The intersection of a set A with the empty set is the empty set.

Complement Rules
A ⋃ A' = U The union of a set with its complement is the universal set.
A ⋂ A' = ∅ The intersection of a set with its complement is the empty set.
U' = ∅ The complement of the universal set is the empty set.
∅' = U The complement of the empty set is the universal set.
(A')' = A The complement of a set's complement is the set itself.

Idempotent Rules
A ⋃ A = A The union of a set with itself is the set itself.
A ⋂ A = A The intersection of a set with itself is the set itself.

Commutative Rules
A ⋃ B = B ⋃ A Union operations are commutative.
A ⋂ B = B ⋂ A Intersection operations are commutative.

Associative Rules
(A ⋃ B) ⋃ C = A ⋃ (B ⋃ C) Union operations are associative.
(A ⋂ B) ⋂ C = A ⋂ (B ⋂ C) Intersection operations are associative.

Distributive Rules
A ⋃ (B ⋂ C) = (A ⋃ B) ⋂ (A ⋃ C) Union operations are distributive over intersections.
A ⋂ (B ⋃ C) = (A ⋂ B) ⋃ (A ⋂ C) Intersection operations are distributive over unions.

DeMorgan Rules
(A ⋃ B)' = A' ⋂ B' The complement of a union between two sets is equal to the intersection of those sets' complements.
(A ⋂ B)' = A' ⋃ B' The complement of an intersection between two sets is equal to the union of those sets' complements.

Sub- and Supersets Under These Rules
A ⊆ B ∨ B ⊇ A ⇒ A ⋂ B = A If A is a subset of B, then the intersection of A and B is A.

Proper Sub- and Supersets Under These Rules
A ⊂ B ∨ B ⊃ A ⇒ A ⋂ B = A ∧ B - A = ∅ If A is a proper subset of B, then the intersection of A and B is A, and additionally the difference of B and A is non-empty.

Properties

  • ℕ, natural numbers:
    • Closed under addition and multiplication.
  • ℤ, integer numbers:
    • Union of ℕ and whole negative numbers.
    • Closed under addition, multiplication, and subtraction.
  • ℚ, rational numbers:
    • ℚ = { x | x = a/b ∧ a,b ∈ ℤ ∧ b ≠ 0 }
    • All numbers that can be expressed as the ratio of two integers, where the divisor is nonzero.
    • All rational numbers have a periodic decimal expansion.
    • Closed under addition, subtraction, multiplication, and nonzero division.
  • ℚ', irrational numbers:
    • Numbers with non-periodic decimal expansions.
  • ℝ, real numbers:
    • Loosely defined as the set of numbers with a 1:1 relationship with evenly spaced points on a straight line.
    • Closed under addition, subtraction, multiplication, and division.
  • ℂ, complex numbers:
    • ℂ ⊂ ℝ
    • Complex numbers are those in which all non-constant polynomials have real roots.
    • ℂ = { x | x = a + bi ∧ a,b ∈ ℝ ∧ i = &sqrt;-1 }
  • 𝔸, algebraic numbers:
    • Any real number that is the solution of a polynomial with integer coefficients of the form

          cnxn + cn-1xn-1 + cn-2xn-2 + ⋯ + c2x2 + c1x1 + c0x0 = 0

      for n > 0 where ci are integer coefficients and cn ≠ 0
  • 𝕋, transcendental numbers:
    • Any irrational number that is not the solution of any polynomial equation with integer coefficients.
    • Examples are π, e, sin(π), cos(2π), tan(π/4)
    • ℝ = 𝔸 ⋃ 𝕋

Properties of Ordering Relations / Inequalities

Given real numbers a and b, their ordering relations may be:

  • a = b
  • a > b
  • a < b
  • a ≥ b
  • a ≤ b

Inequalities of real numbers have these properties:

Reflexivitiy a ≤ a
Trichotomy a < b XOR a = b XOR a > b
Antisymmetry a ≤ b ∧ b ≤ a ⇒ a = b
Transitivity a ≤ b ∧ b ≤ c ⇒ a ≤ c
Addition/Subtraction a ≤ b ⇒ a + c ≤ b + c
a ≤ b ⇒ a - c ≤ b - c
a ≥ b ⇒ a + c ≥ b + c
a ≥ b ⇒ a - c ≥ b - c
Multiplication/Division a ≤ b ∧ c ≥ 0 ⇒ ac ≤ bc
a ≥ b ∧ c ≥ 0 ⇒ ac ≥ bc
a ≤ b ∧ c ≤ 0 ⇒ ac ≥ bc
a ≥ b ∧ c ≤ 0 ⇒ ac ≤ bc

a ≤ b ∧ c ≥ 0 ⇒ a/c ≤ b/c
a ≥ b ∧ c ≥ 0 ⇒ a/c ≥ b/c
a ≤ b ∧ c ≤ 0 ⇒ a/c ≥ b/c
a ≥ b ∧ c ≤ 0 ⇒ a/c ≤ b/c

Given a,b ∈ ℝ ∧ a ≤ b, the following intervals can be defined:

bounded: I1 = (a,b) = { x | a < x < b }
I2 = [a,b] = { x | a ≤ x ≤ b }
I3 = (a,b] = { x | a < x ≤ b }
I4 = [a,b) = { x | a ≤ x < b }
unbounded: I5 = (-∞,b) = { x | x < b }
I6 = [-∞,b] = { x | x ≤ b }
I7 = (a,∞] = { x | a < x }
I8 = [a,∞) = { x | a ≤ x }
I9 = (-∞,∞) = { x | x ∈ ℝ }
degenerate: I10 = { a } (singleton set)
I11 = ∅ (zero length interval)

For the family of all real number intervals, 𝔉, given two intervals I1 and I2:
    I1,I2 ∈ 𝔉 ⇒ I1 ⋂ I2 ∈ 𝔉
    I1,I2 ∈ 𝔉 ∧ I1 ⋂ I2 ≠ ∅ ⇒ I1 ⋃ I2 ∈ 𝔉
    I1,I2 ∈ 𝔉 ∧ I1 ⊄ I2 ∧ I2 ⊄ I1 ⇒ (I1 - I2) ∈ 𝔉

Given two closed, real number intervals A = [aL,aU] and B = [bL,bU], arithmetic operations are:
    C = [aL,aU] + [bL,bU] = [aL + bL, aU + bU]
    C = [aL,aU] - [bL,bU] = [aL - bL, aU - bU]
    C = AB = [min(aL*bL, aL*bU, aU*bL, aU*bU), max(aL*bL, aL*bU, aU*bL, aU*bU)]
    C = A/B = [min(aL/bL, aL/bU, aU/bL, aU/bU), max(aL/bL, aL/bU, aU/bL, aU/bU)]
