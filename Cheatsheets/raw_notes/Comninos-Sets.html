Basic Set Operations and Rules

A ∪ B = { x : x ∈ A ∨ x ∈ B }
    ∴ A ∪ B = B ∪ A
    ∴ A ⊆ A ∪ B
    ∴ B ⊆ A ∪ B

A ∩ B = { x : x ∈ A ∧ x ∈ B }
    ∴ A ∩ B = B ∩ A
    ∴ (A ∩ B) ⊆ A
    ∴ (A ∩ B) ⊆ B

A - B = { x : x ∈ A ∧ x ∉ B }
    ∴ (A - B) ⊆ A
    ∴ (A - B) ∩ (A ∩ B) = ∅
    ∴ (B - A) ∩ (A ∩ B) = ∅
    ∴ (B - A) ∩ (A - B) = ∅

A × B = { (a,b) | a ∈ A ∧ b ∈ B }

∏ni=1Si = S1 × S2 × ⋯ × Sn = { (s1,s2,⋯,sn) | s1 ∈ S1 ∧ s2 ∈ S2 ∧ ⋯ ∧ sn ∈ Sn }

⋃ni=1Si = S1 ∪ S2 ∪ S3 ∪ ⋯ ∪ Sn

⋂ni=1Si = S1 ∩ S2 ∩ S3 ∩ ⋯ ∩ Sn

A ∆ B = { x : (x ∈ A + x ∈ B) }
A ∆ B = { x : (x ∈ A ∧ x ∉ B) ∨ (x ∉ A ∧ x ∈ B) }
A ∆ B = (A ∩ B') ∪ (A' ∩ B)
A ∆ B = (A - B) ∪ (B - A)
A ∆ B = (A ∪ B) - (B ∩ A)
    ∴ A ∆ B = B ∆ A
    ∴ (A ∆ B) ∆ C = A ∆ (B ∆ C)
    ∴ A ∩ (B ∆ C) = (A ∩ B) ∆ (A ∩ C)
    ∴ A ∆ ∅ = A
    ∴ A ∆ A = ∅

A' = Ac = U - A = { x | x ∈ U ∧ x ∉ A }

Theorems on Comparable Sets, given comparable sets A and B

  1. A ⊆ B ⇒ A ∩ B = A
  2. A ⊆ B ⇒ A ∪ B = B
  3. A ⊆ B ⇒ B' ⊆ A'
  4. A ⊆ B ⇒ A ∪ (B - A) = B
  5. A ⊆ B ⇒ (B - A) = (B ∩ A')

Rules of Algebra of Sets

Given universe U and subsets A, B, and C, and rules ∪, ∩, and S':

  • Identity rules:
    • A ∪ ∅ = A
    • A ∩ U = A
    • A ∪ U = U
    • A ∩ ∅ = ∅
  • Complement rules:
    • A ∪ A' = U
    • A ∩ A' = ∅
    • U' = ∅
    • ∅' = U
    • (A')' = A
  • Idempotent rules:
    • A ∪ A = A
    • A ∩ A = A
  • Commutative rules:
    • A ∪ B = B ∪ A
    • A ∩ B = B ∩ A
  • Associative rules:
    • (A ∪ B) ∪ C = A ∪ (B ∪ C)
    • (A ∩ B) ∩ C = A ∩ (B ∩ C)
  • Distributive rules:
    • A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
    • A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
  • DeMorgan rules:
    • (A ∪ B)' = A' ∩ B'
    • (A ∩ B)' = A' ∪ B'
  • Sub- and Super-sets under these rules:
    • A ⊆ B ∨ B ⊇ A ⇒ A ∩ B = A
  • Proper sub and super sets under these rules:
    • A ⊂ B ∨ B ⊃ A ⇒ A ∩ B = A ∧ B - A = ∅

Properties of the Numeric Sets

  • ℝ, real numbers:
    • Loosely defined as the set of numbers with a 1:1 relationship with evenly spaced points on a straight line.
    • Closed under addition, subtraction, multiplication, and division.
  • ℚ, rational numbers:
    • ℚ = { x | x = a/b ∧ a,b ∈ ℤ ∧ b ≠ 0 }
    • All numbers that can be expressed as the ratio of two integers, where the divisor is nonzero.
    • All rational numbers have a periodic decimal expansion.
    • Closed under addition, subtraction, multiplication, and nonzero division.
  • ℚ', irrational numbers:
    • Numbers with non-periodic decimal expansions.
  • ℕ, natural numbers:
    • Closed under addition and multiplication.
  • ℤ, integer numbers:
    • Union of ℕ and whole negative numbers.
    • Closed under addition, multiplication, and subtraction.
  • ℂ, complex numbers:
    • ℂ ⊂ ℝ
    • Complex numbers are those in which all non-constant polynomials have real roots.
    • ℂ = { x | x = a + bi ∧ a,b ∈ ℝ ∧ i = √-1 }
  • 𝔸, algebraic numbers:
    • Any real number that is the solution of a polynomial with integer coefficients of the form

          cnxn + cn-1xn-1 + cn-2xn-2 + ⋯ + c2x2 + c1x1 + c0x0 = 0

      for n > 0 where ci are integer coefficients and cn ≠ 0
  • 𝕋, transcendental numbers:
    • Any irrational number that is not the solution of any polynomial equation with integer coefficients.
    • Examples are π, e, sin(π), cos(2π), tan(π/4)
    • ℝ = 𝔸 ∪ 𝕋

Properties of Ordering Relations / Inequalities

Given real numbers a and b, their ordering relations may be:

  • a = b
  • a > b
  • a < b
  • a ≥ b
  • a ≤ b

Inequalities of real numbers have these properties:

Reflexivitiy a ≤ a
Trichotomy a < b XOR a = b XOR a > b
Antisymmetry a ≤ b ∧ b ≤ a ⇒ a = b
Transitivity a ≤ b ∧ b ≤ c ⇒ a ≤ c
Addition/Subtraction a ≤ b ⇒ a + c ≤ b + c
a ≤ b ⇒ a - c ≤ b - c
a ≥ b ⇒ a + c ≥ b + c
a ≥ b ⇒ a - c ≥ b - c
Multiplication/Division a ≤ b ∧ c ≥ 0 ⇒ ac ≤ bc
a ≥ b ∧ c ≥ 0 ⇒ ac ≥ bc
a ≤ b ∧ c ≤ 0 ⇒ ac ≥ bc
a ≥ b ∧ c ≤ 0 ⇒ ac ≤ bc

a ≤ b ∧ c ≥ 0 ⇒ a/c ≤ b/c
a ≥ b ∧ c ≥ 0 ⇒ a/c ≥ b/c
a ≤ b ∧ c ≤ 0 ⇒ a/c ≥ b/c
a ≥ b ∧ c ≤ 0 ⇒ a/c ≤ b/c

Given a,b ∈ ℝ ∧ a ≤ b, the following intervals can be defined:

bounded: I1 = (a,b) = { x | a < x < b }
I2 = [a,b] = { x | a ≤ x ≤ b }
I3 = (a,b] = { x | a < x ≤ b }
I4 = [a,b) = { x | a ≤ x < b }
unbounded: I5 = (-∞,b) = { x | x < b }
I6 = [-∞,b] = { x | x ≤ b }
I7 = (a,∞] = { x | a < x }
I8 = [a,∞) = { x | a ≤ x }
I9 = (-∞,∞) = { x | x ∈ ℝ }
degenerate: I10 = { a } (singleton set)
I11 = ∅ (zero length interval)

For the family of all real number intervals, 𝔉, given two intervals I1 and I2:
    I1,I2 ∈ 𝔉 ⇒ I1 ∩ I2 ∈ 𝔉
    I1,I2 ∈ 𝔉 ∧ I1 ∩ I2 ≠ ∅ ⇒ I1 ∪ I2 ∈ 𝔉
    I1,I2 ∈ 𝔉 ∧ I1 ⊄ I2 ∧ I2 ⊄ I1 ⇒ (I1 - I2) ∈ 𝔉

Given two closed, real number intervals A = [aL,aU] and B = [bL,bU], arithmetic operations are:
    C = [aL,aU] + [bL,bU] = [aL + bL, aU + bU]
    C = [aL,aU] - [bL,bU] = [aL - bL, aU - bU]
    C = AB = [min(aL*bL, aL*bU, aU*bL, aU*bU), max(aL*bL, aL*bU, aU*bL, aU*bU)]
    C = A/B = [min(aL/bL, aL/bU, aU/bL, aU/bU), max(aL/bL, aL/bU, aU/bL, aU/bU)]
