HTML5 Canvas Cheatsheet

2d Context Methods

Rectangles
  Generic Form Example

rect()

Draw a rectangle.

context.rect(x, y, width, height)

context.rect(20, 20, 150, 100);

fillRect()

Fill a rectangle.

context.fillRect(x, y, width, height)

context.fillRect(20, 20, 150, 100);

strokeRect()

Stroke a rectangle.

context.strokeRect(x, y, width, height)

context.strokeRect(20, 20, 150, 100);

clearRect()

Clear a rectangle inside a given rectangle.

context.clearRect(x, y, width, height)

context.fillStyle="red";
context.fillRect(0, 0, 300, 150);
context.clearRect(20, 20, 150, 100);

Gradients and Patterns
  Generic Form Example

createLinearGradient()

Creates a linear gradient for later use.

context.createLinearGradient(x0,y0,x1,y1)

var grd=ctx.createLinearGradient(0,0,170,0);
grd.addColorStop(0,"black");
grd.addColorStop(1,"white");
ctx.fillStyle=grd;
ctx.fillRect(20,20,150,100);

createPattern()

Repeats a specified element in the specified direction.

context.createPattern(image,
    "repeat|repeat-x|repeat-y|no-repeat")

var img=document.getElementById("lamp");
var pat=ctx.createPattern(img,"repeat");
ctx.rect(0,0,150,100);
ctx.fillStyle=pat;
ctx.fill();

createRadialGradient()

Creates a radial/circular gradient for later use.

context.createRadialGradient(x0,y0,r0,x1,y1,r1)

var grd=ctx.createRadialGradient(75,50,5,90,60,100);
grd.addColorStop(0,"red");
grd.addColorStop(1,"white");
ctx.fillStyle=grd;
ctx.fillRect(10,10,150,100);

addColorStop()

Specifies the colors and stop positions in a gradient object.

gradient.addColorStop(stop,color)

var grd=ctx.createLinearGradient(0,0,170,0);
grd.addColorStop(0,"black");
grd.addColorStop(1,"white");
ctx.fillStyle=grd;
ctx.fillRect(20,20,150,100);

Paths and Lines
  Generic Form Example

fill()

Fills the current drawing path.

context.fill()

ctx.rect(20,20,150,100);
ctx.fillStyle="red";
ctx.fill();

stroke()

Draws a defined path.

context.stroke()

ctx.beginPath(); ctx.moveTo(20,20); ctx.lineTo(20,100); ctx.lineTo(70,100); ctx.strokeStyle="red"; ctx.stroke();

beginPath()

Begins a path or resets current path.

context.beginPath();

ctx.beginPath();
ctx.lineWidth="5";
ctx.strokeStyle="green";
ctx.moveTo(0,75);
ctx.lineTo(250,75);
ctx.stroke();

moveTo()

Moves the path to the specified point, without creating a line.

context.moveTo(x,y)

ctx.beginPath();
ctx.moveTo(0,0);
ctx.lineTo(300,150);
ctx.stroke();

closePath()

Creates a path from current point back to starting point.

context.closePath()

ctx.beginPath();
ctx.moveTo(20,20);
ctx.lineTo(20,100);
ctx.lineTo(70,100);
ctx.closePath();
ctx.stroke();

lineTo()

Adds a new point, creates a line from that point to last specified point.

context.lineTo(x,y)

ctx.beginPath();
ctx.moveTo(0,0);
ctx.lineTo(300,150);
ctx.stroke();

clip()

Clips a region of any shape and size from original canvas.

context.clip()

ctx.rect(50,20,200,120);
ctx.stroke();
ctx.clip();
// Draw red rectangle after clip()
ctx.fillStyle="red";
ctx.fillRect(0,0,150,100);

quadraticCurveTo()

Creates a quadratic Bezier curve.

context.quadraticCurveTo(cpx,cpy,x,y)

ctx.beginPath();
ctx.moveTo(20,20);
ctx.quadraticCurveTo(20,100,200,20);
ctx.stroke();

bezierCurveTo()

Creates a cubic Bezier curve.

context.bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y)

ctx.beginPath();
ctx.moveTo(20,20);
ctx.bezierCurveTo(20,100,200,100,200,20);
ctx.stroke();

arc()

Creates an arc/curve.

context.arc(x,y,r,sAngle,eAngle,counterclockwise)

ctx.beginPath();
ctx.arc(100,75,50,0,2*Math.PI);
ctx.stroke();

arcTo()

Creates an arc/curve between two tangents.

context.arcTo(x1,y1,x2,y2,r)

ctx.beginPath();
ctx.moveTo(20,20);
ctx.lineTo(100,20);
ctx.arcTo(150,20,150,70,50);
ctx.lineTo(150,120);
ctx.stroke();

isPointInPath()

Returns true if specified point is in current path.

context.isPointInPath(x,y)

ctx.rect(20,20,150,100);
if (ctx.isPointInPath(20,50)) { ctx.stroke(); }

Transformations
  Generic Form Example

scale()

Scales current drawing bigger or smaller.

context.scale(scalewidth, scaleheight)

ctx.strokeRect(5,5,25,15);
ctx.scale(2,2);
ctx.strokeRect(5,5,25,15);

rotate()

Rotates current drawing.

context.rotate(angle)

ctx.rotate(20*Math.PI/180);
ctx.fillRect(50,20,100,50);

translate()

Remaps the 0,0 position on the canvas.

context.translate(x,y)

ctx.fillRect(10,10,100,50);
ctx.translate(70,70);
ctx.fillRect(10,10,100,50);

transform()

Replaces current transformation matrix.

context.transform(
    Horizontal-scale,
    Horizontal-skew,
    Vertical-skew,
    Vertical-scale,
    Horizontal-move,
    Vertical-move
)

ctx.fillStyle="yellow";
ctx.fillRect(0,0,250,100);

ctx.transform(1,0.5,-0.5,1,30,10);
ctx.fillStyle="red";
ctx.fillRect(0,0,250,100);

ctx.transform(1,0.5,-0.5,1,30,10);
ctx.fillStyle="blue";
ctx.fillRect(0,0,250,100);

setTransform()

Resets the current transform to the identity matrix, then runs transform().

context.setTransform(
    Horizontal-scale,
    Horizontal-skew,
    Vertical-skew,
    Vertical-scale,
    Horizontal-move,
    Vertical-move
)

ctx.fillStyle="yellow";
ctx.fillRect(0,0,250,100)

ctx.setTransform(1,0.5,-0.5,1,30,10);
ctx.fillStyle="red";
ctx.fillRect(0,0,250,100);

ctx.setTransform(1,0.5,-0.5,1,30,10);
ctx.fillStyle="blue";
ctx.fillRect(0,0,250,100);

Context Properties

  Generic Form Example

Gets/sets color, gradient, or pattern of current fill.

strokeStyle

Gets/sets color, gradient, or pattern of current stroke.

shadowColor

Gets/sets color to use for shadows.

shadowBlur

Gets/sets shadow blur level.

shadowOffsetX

Gets/sets horizontal offset of shadow from shape.

shadowOffsetY

Gets/sets vertical offset of shadow from shape.
